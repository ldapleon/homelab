# Talos Kubernetes Homelab

A production-inspired Kubernetes homelab running on a three-node Proxmox cluster.

## Goals

- Learn Kubernetes from the ground up
- Deploy Talos Linux
- Practice GitOps using Flux CD
- Implement Infrastructure as Code
- Build a highly available home infrastructure
- Migrate Docker workloads to Kubernetes
- Gain hands-on experience with enterprise technologies
